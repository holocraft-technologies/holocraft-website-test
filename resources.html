Frequently Asked Questions

What file formats are supported by HoloCraft displays?

HoloCraft supports a wide range of 3D file formats including OBJ, FBX, STL, GLTF, and USDZ. Our software also includes converters for other formats like 3DS, BLEND, and DAE files.

For optimal performance, we recommend using GLTF or OBJ formats with optimized polygon counts and texture sizes appropriate for real-time rendering.

How do I connect my HoloCraft display to my computer?

HoloCraft displays connect to your computer via USB-C or HDMI, depending on your model. The HoloFan Pro and Enterprise models also support wireless connection via Wi-Fi or Bluetooth for remote control and content streaming.

After connecting, install the HoloCraft Studio software from our website to begin creating and managing your holographic content.

Can multiple users interact with the hologram simultaneously?

Yes, HoloCraft Pro and Enterprise models support multi-user interaction. The system can track and respond to gestures from up to 4 users simultaneously (Pro) or 8 users (Enterprise).

This makes HoloCraft ideal for collaborative environments where multiple team members need to interact with the same holographic content during presentations or design reviews.

What is the viewing angle for HoloCraft holographic displays?

HoloCraft displays offer a 160-degree horizontal viewing angle and a 120-degree vertical viewing angle, allowing content to be visible from most positions around the device.

For optimal viewing and interaction, we recommend positioning users within a 120-degree arc in front of the display at a distance of 1-3 meters, depending on the model size.

How do I create custom gestures for my holographic applications?

Custom gestures can be created using the Gesture Designer tool in HoloCraft Studio. This visual programming interface allows you to define gesture patterns and associate them with specific actions in your application.

For advanced users, our SDK provides a programmatic API for defining complex gesture recognition systems using Python, C#, or JavaScript.
